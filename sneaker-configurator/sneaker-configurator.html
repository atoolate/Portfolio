Ecommerce 3D Sneaker Configurator

A full-stack e-commerce platform featuring an interactive 3D sneaker customization tool paired with a comprehensive admin dashboard for inventory and order management.

Project Overview

This project demonstrates modern full-stack web development by combining immersive 3D graphics with practical e-commerce functionality. Users can customize sneakers in real-time using a 3D configurator, while administrators manage the entire operation through a sophisticated dashboard.
