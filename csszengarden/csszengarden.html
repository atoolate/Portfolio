CSS Zengarden

CSS Zengarden is een open-source CSS-challenge die mensen wilt inspireren en openstellen tot de diverse mogelijkheden van CSS.

Background

Voor deze challenge werkte ik rond het thema futurisme. Ik koos voor een letterlijke weergave van een futuristische zengarden. Daarnaast maakte ik gebruik van CSS grid om de grid-stijl na te bootsen die vandaag vaak terugkomt in modern webdesign zoals bijvoorbeeld bij Apple.


De originele HTML website heeft best veel tekst wat het een uitdaging maakte om een visueel aantrekkelijk design te maken in deze stijl. Ik heb dit trachten op te lossen door te werken met grid-secties en background-images in futuristische stijl. Om in thema te blijven, zijn al deze images custom via AI gegenereerd. Bij hoveren over de secties verschijnt de onderliggende tekst die erbij hoort.
