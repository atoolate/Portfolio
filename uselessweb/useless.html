Useless Web

The useless web is een website met een simpel doel: nutteloze websites samenbrengen. Developers wereldwijd kunnen inzendingen doen en de meest nutteloze website wordt beloond via een link op www.theuselessweb.com

Background

Via Thomas More heeft docent David Heerinckx een eigen versie op poten gezet. Alle studenten konden een inzending doen om gefeatured te worden om de website.


Mijn inzending is een pigfarm. Gebruikers kunnen varkens kweken door met de muis te klikken binnen het veld. Er is echter één klein probleem ...
